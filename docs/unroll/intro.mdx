# Unroll - Build System & Package Manager

**Unroll** is the build system, package manager, and developer tooling ecosystem for Oite. It manages dependencies, handles compilation, and provides a complete developer experience.

## What's in This Section

- **[Design](/unroll/design)** - Architecture and design principles
- **[Modules](/unroll/modules)** - Module resolution and linking

## Overview

Unroll provides:

- **Package Management** - Add, update, and manage dependencies
- **Build System** - Compile projects with optimizations and cross-compilation
- **Developer Tools** - Formatting, linting, type checking, and LSP support
- **Registry** - Public and private package registries

## Architecture

```
┌─────────────────────────────────────────┐
│            User App Code                │
└──────────────────┬──────────────────────┘
                   │
┌──────────────────▼──────────────────────┐
│   Rolls (official system libs)          │
│   @rolls/http, @rolls/tls, @rolls/db    │
└──────────────────┬──────────────────────┘
                   │
┌──────────────────▼──────────────────────┐
│   Unroll (runtime + tooling)            │  <-- THIS SECTION
│   pkg manager, lockfiles, bundler, LSP  │
└──────────────────┬──────────────────────┘
                   │
┌──────────────────▼──────────────────────┐
│   Oite (language core)                  │
│   compiler, type system, ABI, bootstrap │
└─────────────────────────────────────────┘
```

## Quick Start

```bash
# Create new project
unroll new my-app

# Add dependencies
unroll add @rolls/http

# Build project
unroll build --release

# Run project
unroll run
```

## Project Manifest (unroll.toml)

```toml
[package]
name = "my-app"
version = "0.1.0"
edition = "2025"

[dependencies]
"@rolls/http" = "0.1"
"@rolls/json" = "0.1"

[build]
target = "native"
optimization = "release"
```

> **Note**: Unroll is currently in design phase. See the [Design](/unroll/design) document for the planned architecture.
